# Proyecto de Graficación

## Como correr la app (Frontend):

El proyecto estará dividido en dos carpetas, Front y Back, para correr la app en el Frontend.

- Entramos al directorio `/Front`
```bash
cd Front
```
- Una vez dentro de la carpeta /Front, corremos el siguiente comando
```bash
npm run dev 
```

Para entrar en la app del Backend.
- Entramos al directorio `/Back`
```bash
cd Back
```

- Una vez dentro de la carpeta /Back, para correr el servidor se usa el siguiente comando
```bash
npm run dev 
```

Asegurate de tener instalado node y las dependencias del proyecto.

### Instalar las dependencias:
Para instalar las dependencias, igualmente, entra al directorio /Front
```bash
cd Front
```
- Y escribe el comando:
```bash
npm install
```

Para instalar las dependencias en la parte del Backend, del mismo modo se entra al directorio /Back
```bash
cd Back
```

- Y se escriben el siguiente comando:
```bash
npm install
```
